Setup Stripe Webhook
Unzip stripe_webhook.zip file into the root directory of your folder
Go to the stripe_webhook folder and run the following command

```
npm install
```

Install stripe CLI
Login to stripe account
```
stripe login
```

Run the following command to listen to the webhook
```
stripe listen --forward-to localhost:4455/webhook
```
Replcae .env file with your stripe secret key and webhook secret key which is generated by stripe command above

Run the following command to start the server
```
npm run start
```
